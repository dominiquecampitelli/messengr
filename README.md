# Messengr

Interface de chat em tempo real desenvolvida com Next.js (App Router) e React, utilizando Socket.IO Client para comunicação instantânea com o servidor. O front-end oferece uma experiência fluida e responsiva, com layout mobile-first, exibição dinâmica de usuários na sala, indicadores de status e digitação, scroll inteligente de mensagens e componentes reutilizáveis estilizados com Tailwind CSS.

## Como rodar o projeto

Utilizando npm

```bash
npm install
npm run dev
```


O projeto carregará em local http://localhost:3001 ou network http://192.168.15.8:3001.
